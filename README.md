# 💇 Responsive Hair Dresser Website

A modern and stylish **Hair Dresser / Salon Website** built using **HTML, CSS, and JavaScript**.
This website is fully responsive and designed for beauty salons, hairstylists, barbershops, and personal grooming businesses.

---

## ✨ Features

* Responsive Navigation Bar
* Beautiful Home Section
* About Us Section
* Premium Services Showcase
* Hairstyle Gallery
* Pricing Plans
* Blog Section
* Appointment Booking Form
* Customer Reviews
* Social Media Footer
* Mobile Friendly Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons
